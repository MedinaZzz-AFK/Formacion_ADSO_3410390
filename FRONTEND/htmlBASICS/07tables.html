07.tablas

es un elemento que permite organizar y presentar datos estructurados en filas y columnas(datos tabulados). Se utiliza exclusivamente para mostrar información relacional (como horarios, calendarios o estadísticas) y no para el diseño visual de la página web.

Para construir una tabla, se combinan varias etiquetas HTML de forma jerárquica: "table": Es el contenedor principal que define la tabla."tr" (Table Row): Define una fila dentro de la tabla. "th" (Table Header): Define una celda de encabezado. Suele llevar el texto en negrita y centrado. "td" (Table Data): Define una celda de datos normal.
